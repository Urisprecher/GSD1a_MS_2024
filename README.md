# GSD1a_MS_2024
Python and R scripts for "Disease-associated programming of cell memory in glycogen storage disorder type 1a" Manuscript.
This respiratory includes scripts used throughout the manuscript and includes:
* Image-based HCA data analysis- processing & multivariate analysis.
* Imaging & biochemical data processing steps.
* RNA-seq, ATAC-seq, EPIC methylation analysis sequences - from raw data to final analysis. 
* Omics data integration & biochemical data integration.
* Advanced statistics for biological data & batch effect analysis.
